# PicRecog - AI 기반 지능형 이미지 관리 및 시맨틱 검색 시스템

AI를 활용하여 이미지를 자동으로 분류하고, 벡터 데이터베이스(`pgvector`)를 통해 자연어 질의로 유사한 이미지를 지능적으로 검색하는 현대적인 이미지 관리 솔루션입니다.

## 🌟 핵심 기능

- **지능형 이미지 분류 (VLM)**: `Qwen-VL` 기반의 비전 모델을 활용해 7가지 카테고리로 이미지를 정밀하게 자동 분류합니다.
- **에이전트 중심 시맨틱 검색 (Agentic Search)**: `LangChain` 에이전트가 사용자의 자연어 질의를 분석하고, 상황에 맞는 검색 도구를 자율적으로 호출하여 결과를 도출합니다.
- **실시간 비동기 워크플로우**: `LangGraph` 기반의 파이프라인과 `st.fragment`를 결합하여 대량의 이미지 처리 중에도 UI 멈춤 없는 실시간 피드백을 제공합니다.
- **비차단 멀티 태스크 아키텍처**: `UIEventHandler`를 통해 여러 분류/검색 작업을 동시에 실행하고 개별적으로 모니터링할 수 있습니다.
- **에이전트 연동 (MCP)**: `Model Context Protocol` 서버를 내장하여 Claude Desktop 등 외부 AI 에이전트가 PicRecog의 기능을 도구로서 활용할 수 있습니다.
- **고성능 벡터 검색**: 이미지 임베딩과 `pgvector`를 결합하여 시각적/의미적 유사도가 높은 이미지를 고속으로 검색합니다.

## 🚀 실시간 작업 추적 및 UI 가이드

PicRecog은 대규모 데이터 처리 시 사용자 경험을 최우선으로 합니다.

### 📋 이미지 처리 단계

| 단계 | 수행 내용 | UI 상태 표시 |
| :--- | :--- | :--- |
| **1. 로드** | 이미지를 검증하고 분석용으로 최적화합니다. | 작업 바 생성 및 개수 표시 |
| **2. 분석** | AI가 이미지의 내용과 특징을 추출합니다. | 현재 분석 중인 파일명 실시간 스트리밍 |
| **3. 저장** | 결과와 벡터를 DB에 영구 저장합니다. | 진행률 바(Progress Bar) 업데이트 |
| **4. 완료** | 전체 요약 보고 및 작업 목록 정리 기능을 제공합니다. | 완료 상태 및 제거 버튼 활성화 |

### 🚦 작업 상태 표시 (Status)
- 🔵 **RUNNING**: 분석이 활발히 진행 중입니다.
- ✅ **COMPLETED**: 모든 작업이 성공적으로 완료되었습니다.
- ⚠️ **PARTIAL ERROR**: 작업은 끝났으나 일부 파일 처리에 실패했습니다.
- 🔴 **FAILED**: 시스템 오류로 인해 작업이 중단되었습니다.

## 🛠 프로젝트 구조 및 아키텍처 규약

PicRecog은 유지보수와 확장을 위해 엄격한 설계를 따릅니다.

### 📂 폴더 구조
- `backend/`: 핵심 비즈니스 로직 및 AI 클라이언트 (SRP 준수).
- `ui/`: Streamlit 기반 프론트엔드 및 비동기 이벤트 핸들러.
- `shared/`: 프로젝트 전반에서 사용하는 공통 스키마 및 상수.
- `interface/`: MCP 서버 등 외부 시스템 연동 인터페이스.

### 📚 핵심 개발 가이드 (GEMINI.md 참조)
- **자동 의존성 주입**: `@transactional` 데코레이터를 통한 DB 세션 관리 자동화.
- **클라이언트 전문화**: LLM, Embedding 클라이언트의 명확한 역할 분리(SRP).
- **공통 유틸리티**: 중복 방지를 위해 범용 로직은 `backend/system/utils.py`로 집중(DRY).
- **에이전트 호환성**: 로컬 LLM 환경에 최적화된 도구 호출 및 구조화된 응답 파싱 체계 적용.

## 🚀 시작하기

### 1. 전제 조건
- Python 3.12+ (`uv` 권장)
- PostgreSQL (`pgvector` 확장 필요)
- LM Studio 또는 Ollama 호환 API 서버

### 2. 설치 및 실행
```bash
# 의존성 설치
uv sync

# 애플리케이션 실행
uv run streamlit run ui/app.py
```
